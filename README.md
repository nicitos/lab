В рамках темы "Client" реализована клиентская часть (UI) для проекта demo11 с использованием Spring MVC и Thymeleaf. Создан интерфейс для отображения OneToMany отношения (город и погодные записи) через GET-запросы, а также формы для добавления, редактирования и удаления данных. UI разработан с вдохновением от GitHub: таблицы для списков и интуитивные формы.
